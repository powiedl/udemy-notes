NextJS und RQ - Daten beim Laden einer Seite laden (prefetchQuery)

Will man beim Laden einer Seite bestimmte Daten von einer RQ laden, verwendet man dafür prefetchQuery in der Seitenkomponente.

  1. const JobPage = async () => {
  2.   const queryClient = new QueryClient();
  3.   await queryClient.prefetchQuery({
  4.     queryKey: ['jobs'],
  5.     queryFn: () => getAllJobsAction(),
  6.   });
  7.  
  8.   return (
  9.     <HydrationBoundary state={dehydrate(queryClient)}>
  10.       <SearchForm />
  11.       <JobsList />
  12.     </HydrationBoundary>
  13.   );
  14. }

Daneben sollte man dann auch eine loading.tsx (neben der page.tsx) erstellen, die während dem Prefetching angezeigt wird.
