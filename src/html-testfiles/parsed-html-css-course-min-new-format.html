Abschnitt 7: The World of CSS Colors

55. RGBA Colors and Opacity

wenn man opacity als eigenes Attribut im CSS verwendet, gilt es für das gesamte Element - also sowohl für die Textfarbe als auch für die Hintergrundfarbe. Wenn man es direkt (mittels rgba) setzt, gilt es "natürlich" nur dort, wo man es setzt, also entweder bei der background-color oder eben bei der color.

Abschnitt 20: Flexbox: Part 1

171. Working With Flex-Direction

flex-direction legt fest, ob die Main Axis horizontal oder vertikal verläuft:

  • row: main-axis verläuft horizontal (von links nach rechts) (Standardwert) - hängt aber auch von der Einstellung der Schriftrichtung ab (wenn die Schriftrichtung von rechts nach links ist, z. b. Arabisch, verläuft row horizontal - aber von rechts nach links)

  • column: main-axis verläuft vertikal (von oben nach unten)

  • row-reverse: main-axis verläuft horizontal (von rechts nach links)

  • column-reverse: main-axis verläuft vertikal (von unten nach oben)

173. Flex-Wrap

flex-wrap legt fest, ob Elemente in eine neue Zeile (bei flex-direction row) oder eine neue Spalte (bei flex-direction column) wechseln, wenn sie nicht genug Platz haben.

  • nowrap (Standard): keine neue Zeile/Spalte (statt dessen werden die Elemente solange verkleinert, bis sie alle in eine Zeile/Spalte passen)

  • wrap: Elemente kommen in eine neue Zeile/Spalte, wenn sie nicht mehr in eine Zeile/Spalte passen- hier werden sie auch nicht verkleinert.

Dieses Verhalten gilt aber nur, wenn der Container in der entsprechenden Ausdehnung eine Größe festgelegt hat (bei row also width, bei column height). Wenn das nicht der Fall ist, wächst der betreffende Container - wenn er kann (z.b. wenn es max-width oder max-height nicht verhindert), damit alle Elemente hineinpassen (ohne wrap - egal wie die flex-wrap Einstellung ist)

Fortsetzung flex-wrap:

  • wrap-reverse: Elemente kommen in eine neue Zeile/Spalte, wenn sie nicht mehr in eine Zeile/Spalte passen, aber die neue Zeile/Spalte entsteht entgegen der Richtung der zweiten Achse. Bei flex-direction row wird die neue Zeile oberhalb der bisherigen Zeile gemacht (bei wrap unterhalb).
    Bei flex-direction column wird die neue Spalte links der bisherigen Spalte gemacht (bei wrap rechts davon) - jeweils bei Schriftrichtung links nach rechts.
