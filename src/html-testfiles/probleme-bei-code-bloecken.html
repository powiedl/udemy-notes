  10.   }

shadcn Form

Intern verwendet shadcn zod für die Validierung und react-hook-form. Es ist viel Code für das Grundsetup nötig, was sich auszahlt, sobald man etwas umfangreichere Formulare braucht.

Genereller Aufbau:

  1. Man definiert - außerhalb der Formular Komponente - ein formSchema (für die Validierung) - beschreibt die Daten, die erfasst werden:

    1. const formSchema = z.object({
    2.   username: z.string().min(2, {
    3.     message: 'Username min 2 chars',
    4.   }),
    5. });

  2. Man verwendet useForm (von react-hook-form):

    1. const form = useForm<z.infer<typeof formSchema>>({
    2.   resolver: zodResolver(formSchema),
    3.   defaultValues: {
    4.     username: '',
    5.   },
    6. });

  3. Man definiert den onSubmit handler:
    function onSubmit(values: z.infer<typeof formSchema>) { ... }

  4. Und dann definiert man das JSX:

    1. <Form {...form}>
    2.   <form onSubmit={form.handleSubmit(onSubmit)}>
    3.     ...
    4.   </form>
    5. </Form>
