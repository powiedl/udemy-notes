709. Shadcn-ui Form Component and Types

shadcn Form - Fortsetzung

Im Formular verwendet man dann für jedes Feld eine FormField Komponente:

  1. <FormField
  2.   control={form.control}
  3.   name='username'
  4.   render={({ field }) => (
  5.     <FormItem>
  6.       <FormLabel>Username</FormLabel>
  7.       <FormControl>
  8.         <Input placeholder='shadcn' {...field} /> 
  9.       </FormControl>
  10.       <FormMessage />
  11.      </FormItem>
  12.   )}
  13. />

FormMessage wird dabei verwendet gegebenenfalls die Validierungsmeldung für das betreffende Feld darzustellen.
